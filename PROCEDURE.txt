É bem fácil implementar o uso de procedures, vamos lá!

1º Criar a procedure no banco MySQL, para isso, utilize a seguinte estrutura:
DELIMITER //
CREATE PROCEDURE cadastrar_produto (IN produto_nome VARCHAR(255), IN produto_marca VARCHAR(255))
BEGIN
    INSERT INTO produtos (nome, marca) VALUES (produto_nome, produto_marca);
END //
DELIMITER ;

2º Procedure criada, vamos para o Spring, no seu repositório, iremos criar um método para chamar essa procedure:
@Procedure(name = "cadastrar_produto")
    void cadastrarProduto(@Param("produto_nome") String nome, @Param("produto_marca") String marca);

3º Agora é só chamar esse método criado no repositório na sua rota de cadastro. Procure seu controle e implemente a seguinte estrutura:
public void cadastrarProduto(String nome, String marca) {
        produtoRepositorio. cadastrarProduto(nome, marca);
    }
